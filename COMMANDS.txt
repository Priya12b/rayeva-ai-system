# Rayeva AI System - Command Reference

## START BACKEND
cd backend
pip install -r requirements.txt
uvicorn app.main:app --reload

✓ http://localhost:8000/docs

---

## START FRONTEND
cd frontend
npm install
npm start

✓ http://localhost:3000

---

## POPULATE DATABASE
cd backend
python seed_db.py

✓ 10 sample products loaded

---

## TEST MODULE 1 (Category Generator)
curl -X POST "http://localhost:8000/ai/generate-category" \
  -H "Content-Type: application/json" \
  -d '{
    "name": "Bamboo Toothbrush",
    "description": "Eco-friendly bamboo toothbrushes",

## TROUBLESHOOTING
Port 8000 in use?     → uvicorn app.main:app --port 8001
npm not found?        → Install Node.js from nodejs.org
Requirements error?   → pip install --upgrade pip
DB not found?         → Run: python seed_db.py
